# PR本文作成スキル

プルリクエストの説明を以下の形式で日本語で作成する。

## テンプレート

```markdown
## Why

（変更の背景・理由・解決する問題を記述）

## What

（具体的な変更内容を記述）
```

## ガイドライン

- **Why**: なぜこの変更が必要か、どんな問題を解決するかを説明する
- **What**: 何を変更したか、どのような実装をしたかを具体的に記述する
- 箇条書きを使って読みやすくする
- コードの変更だけでなく、意図や背景を伝えることを重視する
