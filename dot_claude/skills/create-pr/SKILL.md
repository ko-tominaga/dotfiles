# PR作成スキル

現在のブランチでプルリクエストを作成する。

## 手順

1. **PR本文を作成する** — `create-pr-body` スキルの形式に従い、Why/What形式の日本語でPR本文を作成する:
   ```markdown
   ## Why
   （変更の背景・理由）

   ## What
   （具体的な変更内容）
   ```

2. **PRを作成する**
   ```bash
   gh pr create --title "<PRタイトル>" --body "<PR本文>"
   ```
   - タイトルは変更内容を簡潔に表す日本語または英語
   - 本文はHEREDOCを使って改行を保持する

## 実行例

```bash
gh pr create --title "feat: ユーザー認証機能を追加" --body "$(cat <<'EOF'
## Why

ログイン機能が未実装で、全ユーザーがアクセスできる状態だったため。

## What

- JWTベースの認証ミドルウェアを追加
- ログイン・ログアウトエンドポイントを実装
EOF
)"
```
